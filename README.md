# EaseCation Config Helper

这是一个用于生成 EaseCation 配置文件的工具。

为了尽可能的轻量化，没有后端，只有前端，可以直接在线使用，不依赖任何服务。

## 功能

1. 商品表配置文件生成
2. 暂时没做哦～

## 运行方法

1. 下载项目到本地

    ```bash
    git clone https://github.com/EaseCation/ec-config-helper.git
    ```

2. 进入项目目录

    ```bash
    cd ec-config-helper
    ```

3. 安装依赖

    ```bash
    yarn install
    ```

4. 运行项目

    ```bash
    yarn dev
    ```
